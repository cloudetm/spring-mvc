spring-mvc

Here is simple Spring MVC demo made by me. Design is done using bootstrap web framework.

It Demonstrates the capabilities of the Spring MVC web framework through small, simple examples. After reviewing this showcase, you should have a good understanding of what Spring MVC can do and get a feel for how easy it is to use. Includes project code.

Here it sperates spring MVC architecture. It has Controller , DAO , JPA , Model ,Repository and Service all separated by different packages. So that you can get a clear idea of code flow. And can understand Spring-MVC with ease.
